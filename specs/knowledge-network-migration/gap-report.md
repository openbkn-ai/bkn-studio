## 当前主要差异

### 结构已到位，但视觉仍未完全对齐
- 知识网络列表页工具栏与旧 `vega` 的最终观感仍有明显差异，尤其是搜索框、标签筛选、排序/刷新图标组
- 工作区左侧导航、内容区留白、卡片密度还没有做第一轮统一收口
- 各资源页的工具栏、列表密度、详情层级还不够统一

### 仍是第一阶段占位或简化
- 知识网络导入仍是占位
- 概念分组导入仍是占位
- 对象类创建 / 编辑仍是简化版，不是旧 `vega` 三步式表单
- 关系类创建 / 编辑仍是简化版，不是旧 `vega` 的完整映射配置
- 行动类创建 / 编辑仍是简化版，不含执行映射与计划
- 预览仍不是旧 `vega` 的完整图谱体验

### 尚未迁入
