# 知识网络视觉与交互基线

## 1. 工作区层级

- 一级菜单进入知识网络列表
- 知识网络卡片进入工作区
- 工作区必须保持：返回区 + 标题区 + 左侧二级导航 + 主内容区

## 2. 左侧二级导航

- 导航项顺序要稳定
- 当前阶段最少包含：
  - 概览
  - 预览
  - 概念分组
  - 对象类
  - 关系类
  - 行动类
- 选中态要明显，不能只靠微弱背景差异

## 3. 列表资源页统一规则

- 页面内部保持“工具栏 + 列表区”结构
- 左侧是主操作按钮，右侧是查询/筛选区或提示区
- 列表项标题区必须有：
  - 颜色标识或图标
  - 名称
  - 标签
- 行尾必须有：
  - 更新时间
  - 更新人
  - 操作入口

## 4. 详情页统一规则

- 详情页必须按信息分组展示，不能直接平铺字段
- 至少包含：
  - 基础信息
  - 关联信息
  - 扩展信息/状态

## 5. 当前阶段允许的视觉妥协

- 不强求旧 `vega` 的像素级 1:1
- 允许不用旧项目私有图标系统
- 允许不用旧项目私有表格封装

## 6. 当前阶段不允许的偏差

- 工作区层级消失
- 左侧二级导航缺失
- 列表页工具栏和列表区关系混乱
- 弹窗/抽屉信息层级不清晰
- 空态和错误态缺失
